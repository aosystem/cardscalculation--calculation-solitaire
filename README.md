# Playing Cards Score Calculator 🃏🔢

A professional score management tool for card games built with **Flutter**. This app simplifies complex point calculations, allowing players to focus on the game strategy rather than manual arithmetic.

## 🌟 Key Features
- **Effortless Calculation:** Quickly add and track scores for multiple players.
- **Game History:** Keep a detailed record of scores throughout the match.
- **Global Accessibility:** Fully localized in **50+ languages**, making it a perfect companion for international game nights.
- **Cross-Platform:** High-performance UI/UX powered by Flutter for both Android and iOS.

## 💡 Best For...
- **Card Game Enthusiasts:** Managing scores for various games like Rummy, Bridge, or Poker variants.
- **Game Nights:** Eliminating the need for pen and paper for scorekeeping.
- **Tournaments:** Fast and reliable data entry for competitive play.

## 🛠 Technical Overview
- **Framework:** Flutter
- **Language:** Dart
- **Logic:** Robust state management for score tracking and historical data persistence.

*Note: This repository contains the core logic and source code. Assets (images, sound effects, and localization files) are excluded to protect intellectual property.*
